# Garage Sale Sales Recorder

A small static browser app for recording yard sale purchases into buckets A, T, and Z.

## Run locally

Open `index.html` in a browser.

## Deploy

Deploy this folder to Vercel as a static site. The app stores sales in `localStorage`, so data stays only on the device and browser that recorded it.

## Notes

- Intake records bucket, amount, date, and timestamp.
- Report groups totals by day and bucket.
- Reset clears only the current browser's local copy.
